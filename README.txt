To set up the project:
    Unzip the project into a folder you can access with whatever you use to compile and run java
    Make sure the files HashTable.java, LinkedList.java, Product.java, RedBlackBinaryTree.java, RetailerDriver.java, and User.java are accessible to each other
To run the project:
    Use whatever method of compiling and running java code you use to run RetailerDriver.java
To test the project:
    Interact with the console when prompted and test out the options
